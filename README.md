[vTaskr Backend](https://github.com/Valbou/vtaskr-backend)

Made with Svelte 5

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Run test suite

```sh
npm run test
```

### Compile and Minify for Production

```sh
npm run build
```
